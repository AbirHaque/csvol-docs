Commands

Importable by writing the following at the top of your main.csvol file: 

    IMPORT COMMAND csvol-sol [item]

Accessable by writing the following: 

    IMPORTED [item] [args...]

Available Commands:

Delimiter
Sets custom delimiters.

Display
Generates spreadsheet GUI of CSV dataset. Can edit 1-2 row CSV datasets. Will make it possible to edit larger CSV datasets in GUI in future versions.
