Commands

Importable by writing the following at the top of your main.csvol file:
 IMPORT COMMAND csvol-sol [item]

Accessable by writing the following:
 IMPORTED [item] [args...]
Delimiter
Sets custom delimiters.

Display
Generates spreadsheet GUI of CSV file. Can edit 1-2 row CSV files. Will make it possible to edit larger CSV files in GUI in future versions.


Modules

Importable by writing the following at the top of your main.csvol file:
 IMPORT COMMAND csvol-sol [item]

Accessable by writing the following:
 PULL MODULE [item]
Soon to be added.

Files

Importable by writing the following at the top of your main.csvol file:
 IMPORT FILE csvol-sol [item]
