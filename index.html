About

Character Separated Value Organizer Language, or CSVOL (\kasväl\), is an extensible scripting language designed to interact with online program libraries. CSVOL is intended for programmers, both new and experienced, who don’t want to waste time compiling code for small programs.

CSVOL 21 is the newest version of CSVOL. Screenshots of CSVOL 21 are soon to come. Videos on CSVOL 21 and SOL's progress are on Abir Haque's YouTube channel .

Current Features

Asides from baseline features like creating/editing/printing file contents, looping, conditional statements, and math operators, the current features of CSVOL include

- Importable online libraries (SOL)
- Online file downloading/syncing
- Additional REPL environment
- Speech synthesis (with the CSVOL Voice library)
- GUI spreadsheet (with the CSVOL Display library)
- CSVOL script modularization
- File name and line number error reporting
