About

Character Separated Value Organizer Language, or CSVOL (\kasv√§l\), is a language that specializes in reporting and manipulating delimited files in both GUI and text format.

Current features

- Invocable REPL environment
- Importable libraries (SOL)
- Editable GUI spreadsheet
- Modularization
- Create, add, edit, delete, and print file contents
- For loops
- Conditional statements
- Simple math operators

Built with

- The interpreter and SOL, or Standard Online Library, are written in Java.
- The SOL host website was built with HTML. The SOL is present here: https://csvol-docs.wichitacpu.repl.co/sol.html.

All development occurred on REPL.it, with backups stored on GitHub.
